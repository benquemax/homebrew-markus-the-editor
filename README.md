# Homebrew Tap for Markus

Homebrew cask formula for [Markus](https://github.com/benquemax/markus-the-editor) - a local WYSIWYG Markdown editor.

## Installation

```bash
brew tap benquemax/markus-the-editor
brew install --cask markus
```

## Updating

```bash
brew upgrade --cask markus
```

## Uninstalling

```bash
brew uninstall --cask markus
brew untap benquemax/markus-the-editor
```
